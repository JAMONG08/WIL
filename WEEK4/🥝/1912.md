# 공통문제 1912 : (재귀함수) 팩토리얼 계산
- [문제 바로가기](https://codeup.kr/problem.php?id=1953)

### 문제
팩토리얼 계산
##### 입력 예시
> 5
##### 출력 예시
> 120


### 문제 코드 구현(python)
```python

# 팩토리얼 (재귀함수)
def factorial(n):
  if(n == 1):
    return 1
  return n * factorial(n-1)

# 입력값 받기
n = int(input())

result = factorial(n)
print(result)

```
